# Digital Copyright Vault

This repository allows you to generate signed, dated, and hashed copyright certificates manually via GitHub Actions.

## Setup
1. Upload your signature image as `signature.png` to the root folder.
2. Create two folders: `designs/` and `output/`.

## How to Generate a Certificate:
1. Go to the **Actions** tab in this repository.
2. Click on **Generate Copyright PDF** on the left.
3. Click the **Run workflow** dropdown button.
4. **Enter your name manually** in the "Owner Name" field.
5. Enter the title of your design.
6. Click **Run workflow**.
7. After 1 minute, your signed PDF will appear in the `output/` folder.
